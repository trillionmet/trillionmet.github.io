Project Planning

Project planning is part of project management, which relates to the use of schedules such as Gantt charts to plan and subsequently report progress within the project environment..

  • Project planning can be done manually or by the use of project management software.
  • A project plan, according to the Project Management Body of Knowledge,.
  • A formal, approved document used to guide both project execution and project control.
  • Timelines and Gantt charts for key milestones.
  • Communication plans.
  • Resources need to complete the project, like project management tools, cash, freelancers, and more
